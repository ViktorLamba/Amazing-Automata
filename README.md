# Amazing-Automata
Универсальное решение

В нашем проекте пока что реализованиы 4 языка, а именно:
- Rust
- Python
- Golang
- C++

Конкретная инструкция по каждому

# Rust
Обязательный файлы для корректной работы:
"Cargo.toml" - файл со всеми зависимостями
"main.rs" или "lib.rs" или любой другой файл с расширением ".rs" - точка входа в приложение

# Python
Обязательный файлы для корректной работы:
"pyproject.toml" либо "requirements.txt" - файл со всеми зависимостями
"main.py" либо любой другой файл с расширением ".py" - точка входа в приложение

# GO
Обязательный файлы для корректной работы:
"go.mod" - файл со всеми зависимостями
"main.go" или любой файл с расширением ".go" - точка входа в приложение

# C++
Обязательный файлы для корректной работы:
"CMakeLists.txt" или "Makefile" - файл со всеми зависимостями
"main.cpp" ,"main.cc" ,"main.cxx" либо файлы с расширением ".cpp",".cpp" ,".cxx" ,".c++" - точка входа в приложение
